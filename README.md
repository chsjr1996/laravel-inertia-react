# Laravel Inertia React
This repository contains an simple Laravel application with Inertia.JS and React template.

## Features

- PHP 8.2+ support
- Laravel 10
- Laravel Vite 4
- React 18
- Inertia.JS 1.0
- Prettier/Eslint support ⚙️
